Kimi 的暴涨验证了关键假设：Agent 的 Token 消耗是普通聊天的 100-1000 倍，一旦用户从"聊天"转向"用 Agent 办事"，付费意愿和客单价指数级跳升。Stripe 排名全球第 9 意味着 Kimi 已是全球 AI 订阅收入的头部玩家。AI 商业化核心逻辑转变——从"卖模型能力"到"卖 Agent 完成任务的价值"。用户为聊天付费天花板低（$20/月），但为 Agent 办事付费天花板极高（数百至数千美元/月）。启示：① Token 消耗量级跳变，云服务商最确定受益；② Agent竞争维度转为"谁能更可靠地完成更多任务"；③ 中国 AI 全球化路径打通——Kimi 通过 Stripe 直接面向全球收费，绕过本地化壁垒。谁让 Agent 从"能用"变成"离不开"，谁就能复制这条曲线。
开源 Karpathy 开源 autoresearch：630 行代码 AI 自主研究员，单卡一夜跑百次实验
Andrej Karpathy 开源 autoresearch——将 LLM 训练压缩为约 630 行单文件代码，单 GPU 即可运行。AI Agent 自主循环"编辑代码→训练 5 分钟→检查得分→保留/回滚"，无需人工干预。下一阶段：从单 Agent 进化为多 Agent 异步协作，模拟"由博士生组成的研究社区"。Karpathy 本人正在 8×H100 上运行更大规模版本。
代码量 630行 单次训练 5分钟 生产环境 8×H100
来源：GitHub · AI寒武纪 ｜ 3月8-9日
💡 深度洞察
核心突破不在代码量（630行），而在证明了AI 科研的"最小闭环"：提出假设→编写实验→跑数据→评估→迭代，全程无人干预。传统科研一轮循环数周，这套系统单 GPU 一夜可跑百次。下一步：多 Agent 异步协作，模拟"博士生研究社区"——多个 Agent 各自探索不同假设空间，共享发现，交叉验证。速度快数千倍。药物研发、材料科学、芯片设计等需要大量实验迭代的领域将率先受益。Karpathy 在 8×H100 上运行更大版本说明算力投入回报正从"训练更大模型"转向"用模型自动做科研"。科学发现的速度，将取决于算力而非人力。
