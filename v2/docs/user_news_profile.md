## 三、内容偏好详解

### 3.1 AI模型与产品（第一优先级）

| 类型 | 具体特征 | 典型案例 |
|------|----------|----------|
| **模型战略调整** | 大厂产品方向变化、商业策略转向 | OpenAI关闭Sora、Pre-IPO文件泄露 |
| **AI Agent落地** | 智能体产品发布、功能更新 | Claude Code Auto Mode、火山引擎百万亿Token |
| **模型性能突破** | 新模型发布、Benchmark成绩 | GPT-4.5、Gemini 2.5 Pro、DeepSeek-V3 |
| **开源生态** | 开源模型、社区动态 | HuggingFace趋势、Llama更新 |

### 3.2 芯片与算力（第一优先级）

| 类型 | 具体特征 | 典型案例 |
|------|----------|----------|
| **产能/良率** | 晶圆厂产能、良率数据 | 台积电亚利桑那厂良率追平台湾 |
| **GPU新品/订单** | NVIDIA新品、订单动态 | B300产能爬坡、GB200、Blackwell架构 |
| **价格动态** | 内存、GPU价格波动 | 内存价格暴涨51%、HBM产能紧张 |
| **国产芯片突破** | RISC-V、AI芯片、性能里程碑 | 阿里玄铁C950、平头哥真武810E涨价 |
| **供应链分析** | 地缘政治、产能转移 | 美国4万亿芯片联盟、中美AI GPU差距 |

### 3.3 手机与消费电子（第一优先级 - 全平台均衡）

**核心原则：苹果、三星、Google、华为、小米等全平台均衡关注，不偏向单一品牌。**

| 品牌 | 关注内容 | 典型信源 |
|------|----------|----------|
| **Apple** | iOS更新、硬件战略、生态布局 | 9to5Mac、AppleInsider、Bloomberg |
| **Samsung** | Galaxy系列、折叠屏、AI功能、GaN芯片 | Samsung Newsroom、SamMobile |
| **Google** | Android更新、Pixel、AI原生功能、搜索博弈 | 9to5Google、Google Blog |
| **华为** | 鸿蒙生态、麒麟芯片回归、全场景 | 微博/机器之心、华为官方 |
| **小米** | 手机AI、人车家生态、澎湃芯片 | 微博/竞核、小米官方 |
| **OPPO/vivo/一加** | 新品、技术创新、IP联名 | 微博、中文科技媒体 |
| **其他** | 亚马逊、传音等新动态 | 综合科技媒体 |

**选题平衡原则：**
1. 每篇日报手机模块至少覆盖 **3个不同品牌**
2. 同一品牌连续两天头条需有 **差异化角度**（产品/战略/技术）
3. 国产厂商与国际品牌比例保持 **4:6 或 5:5**
4. 重大发布（折叠屏、AI功能）优先覆盖，不受品牌限制

### 3.4 游戏行业（第二优先级 - 专业关注）

**重要说明：对游戏行业持专业关注态度，重视平台商业策略和行业动态，非纯娱乐内容。**

| 类型 | 具体特征 | 典型案例 |
|------|----------|----------|
| **平台商业动态** | 促销、定价、订阅策略 | **Xbox春季促销（重要节点）**、PSN季度促销 |
| **硬件销量数据** | 主机销量、生产调整 | Switch 2销量超越PS5、减产30% |
| **行业重大事件** | 裁员、收购、战略调整 | Epic裁员1000人、沙特持股Capcom 10% |
| **技术争议** | AI素材、技术问题、玩家反弹 | 红色沙漠AI素材争议、卡普空表态 |
| **游戏+AI交叉** | AI NPC、生成式内容 | AI游戏角色、AI辅助开发 |

**弱偏好（易淘汰）：**
- 纯娱乐/文化内容（游戏世界地图测试、插画版等）
- 缺乏产业/商业洞察的内容

### 3.5 科技行业动态（第二优先级）

| 类型 | 具体特征 | 典型案例 |
|------|----------|----------|
| **大厂财报** | 营收、增速、市场份额 | 字节1500亿美元、OpenAI 9亿用户 |
| **战略调整** | 业务收缩、重组、IPO | OpenAI战略收缩、Pre-IPO文件 |
| **人事变动** | 高管变动、关键人才流动 | 重要AI研究员跳槽 |
| **并购投资** | 收购、投资动态 | 沙特投资Capcom、微软收购案 |

### 3.6 政策与监管（第二优先级）

| 类型 | 具体特征 | 典型案例 |
|------|----------|----------|
| **地缘政治** | 出口管制、技术封锁 | 美国AI芯片出口管制、欧盟DMA |
| **行业监管** | 反垄断、数据隐私 | 欧盟AI法案、中国AI监管 |
| **产业补贴** | 芯片法案、政府投资 | 美国4万亿芯片联盟、各国半导体补贴 |

## 七、淘汰规则

| 类型 | 淘汰原因 | 典型案例 |
|------|----------|----------|
