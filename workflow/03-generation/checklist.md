# 日报生成检查清单

## 生成前检查

- [ ] 已读取人工确认的 `daily_selection_YYYYMMDD.md`
- [ ] 已读取 `template.html` 模板
- [ ] 已读取 `user_news_profile.md` 偏好画像

## 内容检查

- [ ] 时效性：全部新闻在48h内，主体在24h内
- [ ] 数量：每板块 ≥ 5条新闻
- [ ] 信源：官方一手源 ≥ 5条
- [ ] 信源：同一二手源整期引用 ≤ 2次
- [ ] 深度：至少3-5篇全文抓取
- [ ] 洞察：基于全文+权威观点，非仅凭摘要

## 格式检查

- [ ] 基于 `template.html` 填充（非自行创建）
- [ ] 所有链接可点击（使用原始 source.url）
- [ ] 标题格式正确
- [ ] 数据标签完整
- [ ] 深度洞察 ≤300字符

## 输出检查

- [ ] 文件保存到 `brief/YYYY-MM-DD.html`
- [ ] 文件大小正常（30KB-100KB）
- [ ] 可在浏览器正常打开

## 发布后检查

- [ ] GitHub Pages 已更新
- [ ] 测试邮件已发送
- [ ] 群发邮件已发送
- [ ] 公众号已同步
